# Graphite

Dependency graph similar to nx graph for golang.
Build the project then invoke it from desired directory or pass a path as args.

## Getting Started

Clone the repository and run below to build executable.
```bash
make build
```

## Usage

built package run using
```bash
graphite <path_to_go_module>
graphite .
```

## Requirements

To build the Go 1.25 is required.
